# Claude Code Configuration

- **Thesis Title**: Nghiên cứu mô hình ngôn ngữ lớn tăng cường đồ thị tri thức trong dự báo tài chính và lập luận đầu tư dựa trên ngữ cảnh
- **Research Focus**: Multi-Agent Trading Framework with LLMs and Knowledge Graphs
- **Language**: Vietnamese (thesis), English (code)

## Key Directories
- `src/` - Main source code
- `research/` - Literature, research notes, and findings
- `data/` - Datasets and data processing
- `tests/` - Unit and integration tests
- `docs/` - Project documentation and technical specs
- `document/` - Thesis documents and reference papers

## Development Workflow
- Create feature branches for new research/development
- Keep main branch stable
- Document decisions in `docs/decisions/`
- Add research findings to `research/notes/`

## Code Standards
- Python for ML/data work
- Clear naming conventions (English)
- Type hints for Python code
- Document complex algorithms

## Review Checklist
- Code review before merging
- Tests pass locally
- Documentation updated
- No sensitive data committed
