# Research Materials

This directory contains research-related materials and findings.

## Structure

- **papers/** - Reference papers and PDFs (also in document/paper/)
- **notes/** - Research notes and analysis
- **findings/** - Key findings and discoveries

## Adding Research Notes

When adding new research findings:
1. Create a markdown file in `notes/` with descriptive name
2. Include source/reference
3. Summarize key points
4. Link to implementation in code

Example:
```markdown
# Title: [Research Topic]
Date: 2026-04-11
Source: [Paper/Article Name]

## Summary
[Brief summary]

## Key Points
- Point 1
- Point 2

## Implementation Notes
- Related code in `src/...`
```
