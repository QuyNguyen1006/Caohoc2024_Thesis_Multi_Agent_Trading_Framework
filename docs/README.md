# Documentation

This directory contains project documentation.

## Structure

- **decisions/** - Architecture Decision Records (ADRs)
- **architecture/** - System architecture and design documents
- **api/** - API documentation

## Getting Started

See the project README for setup instructions.

## Contributing

When making significant decisions, document them in `decisions/` as ADRs.

### ADR Template
```markdown
# ADR-001: [Title]

## Status
Proposed | Accepted | Deprecated

## Context
[Description of the issue or problem]

## Decision
[Description of the chosen approach]

## Consequences
[Pros and Cons]
```
